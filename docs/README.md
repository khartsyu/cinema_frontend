В своей дипломной работе мною были использованы следующие технологии:\
В верстке HTML5	- структура страниц;\
Стили	SCSS -	удобная работа с CSS, переменные, вложенность, миксины;\
Сетку	Bootstrap подключила, но не использовала, отдала предпочтение SCSS;\
Методология	БЭМ	- именование классов для читаемости и переиспользования;\
Логика JavaScript (ES6+)	- динамика, работа с API, бронирование;\
Работа с API	Fetch API - отправка и получение данных с сервера;\
QR-коды	QRCreator.js - генерация QR-кодов для билетов;\
Хранение данных	localStorage - Передача данных между страницами (выбранный сеанс, места);\
Система контроля версий	GitHub	- хранение кода, управление версиями;\
GitHub Pages	- публикация сайта в интернете.
